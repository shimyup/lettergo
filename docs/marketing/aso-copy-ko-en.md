### Korean
- App Name
  - LetterGo: 글로벌 편지 친구
- Subtitle / Short Description
  - 지구 반대편으로 보내는 진심 한 통
- Long Description (Draft)
  - LetterGo는 전 세계 사용자와 편지를 주고받는 감성 소셜 앱입니다.
  - 편지는 지도 위를 실제로 이동하며, 도착까지의 시간이 설렘을 만듭니다.
  - 익명 기반으로 부담 없이 시작하고, 답장과 수신을 통해 자연스럽게 관계를 쌓을 수 있습니다.
  - 주요 기능:
  - 세계 지도 기반 편지 배송 여정
  - 국가 랜덤 발송과 근처 도착 알림
  - 받은 편지함, 답장, DM
  - 프리미엄: 발송량 확장, 이미지/링크 첨부, 특급 배송

### English
- App Name
  - LetterGo: Global Pen Pals
- Subtitle / Short Description
  - Send heartfelt letters across the world
- Long Description (Draft)
  - LetterGo is an emotional social app where you exchange letters with people worldwide.
  - Each letter travels on the map, turning delivery time into anticipation.
  - Start safely with anonymous-first interactions, then build real connections through replies.
  - Key features:
  - Map-based global letter journey
  - Random country sending and nearby arrival alerts
  - Inbox, replies, and DM
  - Premium: higher limits, image/link letters, express delivery
